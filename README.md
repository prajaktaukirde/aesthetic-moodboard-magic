# AestheticX AI

## Tech Stack

- **Frontend Framework**: React 18 with React Router, React Hook Form
- **UI Toolkit**: Radix UI components, shadcn/ui patterns, Tailwind CSS, Lucide React
- **Build Tool**: Vite 8
- **Language**: TypeScript 5
- **State Management**: TanStack React Query
- **Validation**: Zod
- **Testing**: Vitest, Playwright
- **Package Manager**: npm / bun
